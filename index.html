Experiência

Tenho habilidade comprovada para atuar desde o planejamento inicial até a entrega do produto final, agregando valor tanto em aspectos estratégicos quanto operacionais. Minha experiência diversificada inclui trabalhar com uma ampla gama de tecnologias, bancos de dados, serviços em nuvem e frameworks específicos para o desenvolvimento web. Estou comprometido em criar soluções inovadoras e funcionais que atendam às necessidades dos clientes e ofereçam uma experiência de usuário excepcional. Trabalho como desenvolvedor há mais de 5 anos e utilizo as mais diversas tecnologias de programação e aplicativos web como Django, Python, HTML5, CSS3, JavaScript, Bootstrap, Azure DevOps (CI/CD, pipeline), BitBucket, SonarCloud, RobotFramework, MySQL, SQlite3, entre outros, para o desenvolvimento de aplicações web.
